# 用户信息管理平台

一个基于 **Python Flask** 构建的简易用户信息管理系统，提供登录、用户信息展示、登出等基础功能。

## 📋 功能特性

- ✅ 用户登录认证（密码哈希存储、安全比对）
- ✅ 用户信息展示（用户名、邮箱、手机、角色、余额）
- ✅ 用户登出
- ✅ Session 会话管理
- ✅ 安全的密码存储（`werkzeug.security` 哈希加密）
- ✅ 响应式卡片式 UI 界面

## 🛠️ 技术栈

| 技术 | 说明 |
|------|------|
| **Python 3** | 编程语言 |
| **Flask** | Web 框架 |
| **Werkzeug** | 密码哈希与安全工具 |
| **HTML5 + CSS3** | 前端页面与样式 |

## 🚀 快速开始

### 环境要求

- Python 3.8+
- pip（Python 包管理器）

### 安装步骤

1. **克隆仓库**

```bash
git clone https://github.com/guikkttyy/gg1.git
cd gg1
```

2. **安装依赖**

```bash
pip install flask
```

3. **启动服务**

```bash
python app.py
```

4. **访问系统**

打开浏览器访问：

```
http://127.0.0.1:5000
```

## 🌐 路由说明

| 路由 | 方法 | 功能 |
|------|------|------|
| `/` | GET | 首页，已登录则展示用户信息，未登录则提示登录 |
| `/login` | GET/POST | 登录页面，POST 提交用户名密码进行认证 |
| `/logout` | GET | 清除 session，退出登录并跳转到首页 |

## 🧩 页面展示

### 登录页
- 卡片式居中布局
- 用户名 / 密码输入框
- 登录按钮
- 错误提示（用户名或密码错误）

### 首页
- **已登录状态**：显示"欢迎回来" + 用户完整信息（用户名、邮箱、手机、角色、余额）+ 退出按钮
- **未登录状态**：显示"请先登录" + 跳转登录按钮

### 导航栏
- 左侧品牌名称：用户管理系统
- 右侧菜单：登录状态显示"欢迎，用户名" + "退出" | 未登录显示"登录"
- 蓝色渐变背景（`#667eea` → `#764ba2`）

## 🔒 安全特性

| 安全措施 | 说明 |
|----------|------|
| 🔐 密码哈希存储 | 使用 `generate_password_hash()` 对密码进行哈希加密 |
| ✅ 安全密码比对 | 使用 `check_password_hash()` 进行哈希比对，避免时序攻击 |
| 🛡️ 不暴露密码 | 密码字段不会传递到前端模板渲染 |
| 🔑 随机 Secret Key | 优先从环境变量 `SECRET_KEY` 读取，否则使用 `os.urandom(24)` 随机生成 |
| 🔒 Session 管理 | 登录状态存储在 Flask session 中，登出时清除 |

## ⚙️ 环境变量

| 变量 | 说明 | 默认值 |
|------|------|--------|
| `SECRET_KEY` | Flask session 加密密钥 | 随机生成（`os.urandom(24).hex()`） |
