# 用户信息管理平台

一个基于 **Python Flask** 构建的用户信息管理系统，提供登录、注册、用户信息展示、用户搜索及登出等完整功能。

---

## 📋 功能特性

| 功能 | 说明 |
|------|------|
| ✅ **用户登录** | 账号密码认证，密码哈希安全存储 |
| ✅ **用户注册** | 新用户自助注册，数据存入 SQLite 数据库 |
| ✅ **用户搜索** | 按用户名或邮箱模糊搜索，结果以表格展示 |
| ✅ **信息展示** | 登录后展示个人完整信息（用户名、邮箱、手机、角色、余额） |
| ✅ **用户登出** | 清除会话，安全退出 |
| ✅ **Session 管理** | Flask session 维护登录状态 |
| ✅ **SQL 注入防护** | 所有 SQL 使用参数化查询，杜绝注入风险 |

---

## 🛠️ 技术栈

| 技术 | 用途 |
|------|------|
| **Python 3** | 编程语言 |

## 🌐 路由说明

| 路由 | 方法 | 功能 | 说明 |
|------|------|------|------|
| `/` | GET | 首页 | 已登录 → 展示用户信息；未登录 → 提示登录 |
| `/login` | GET / POST | 登录 | POST 提交用户名密码验证，哈希比对 |
| `/register` | GET / POST | 注册 | POST 提交信息写入 SQLite（参数化查询） |
| `/search` | GET | 搜索用户 | 参数 `?keyword=xxx`，模糊匹配用户名/邮箱 |
| `/logout` | GET | 登出 | 清除 session 后重定向到首页 |

## 🧩 页面展示

### 导航栏
- **左侧**：品牌名称「用户管理系统」
- **右侧（未登录）**：登录 · 注册
- **右侧（已登录）**：欢迎，用户名 · 退出
- 蓝色渐变背景（`#667eea` → `#764ba2`）

### 登录页 `/login`
- 卡片式居中布局
- 用户名 / 密码输入框 + 登录按钮
- 错误提示（密码错误）
- 注册成功时显示绿色提示「注册成功，请登录」

### 注册页 `/register`
- 卡片式布局
- 用户名、密码、邮箱、手机号输入框 + 注册按钮
- 注册失败显示错误信息

### 首页 `/`
- **已登录**：
  - 欢迎语 + 用户详细信息列表
  - 搜索框：输入关键词模糊搜索已注册用户
  - 搜索结果以表格展示（ID / 用户名 / 邮箱 / 手机号）
  - 无结果时显示「无搜索结果」
  - 退出登录按钮
- **未登录**：
  - 提示「请先登录」+ 跳转登录按钮

## 🗄️ 数据存储架构

本项目采用 **双数据源** 架构：

| 数据源 | 用途 | 存储位置 |
|--------|------|----------|
| **USERS 字典（内存）** | 登录认证 | `app.py` 中硬编码 |
| **SQLite 数据库** | 用户注册 + 搜索查询 | `data/users.db` |

> 登录使用内存字典中的用户（密码已哈希），注册用户存入 SQLite 数据库，搜索功能查询 SQLite。

## 🔒 安全措施

| 安全措施 | 实现方式 |
|----------|----------|
| 🔐 **密码哈希存储** | `generate_password_hash()` 对密码加密 |
| ✅ **安全密码比对** | `check_password_hash()` 哈希比对，防时序攻击 |
| 🛡️ **密码不暴露** | 密码字段不传递到前端模板 |
| 🔑 **随机 Secret Key** | 优先 `SECRET_KEY` 环境变量，否则 `os.urandom(24)` |
| 🔒 **Session 管理** | 登录状态存 session，登出时清除 |
| 🚫 **SQL 注入防护** | 所有 SQL 使用 `?` 占位符 + 参数元组执行 |

---

## ⚙️ 环境变量

| 变量 | 说明 | 默认值 |
|------|------|--------|
| `SECRET_KEY` | Flask session 加密密钥 | 随机生成（`os.urandom(24).hex()`） |

## 🧪 SQL 注入演示说明（仅供学习）

> ⚠️ 当前项目已修复所有 SQL 注入漏洞。

本项目早期版本在以下两个接口存在 SQL 注入漏洞，现已通过**参数化查询**修复：

| 接口 | 原始问题 | 修复方式 |
|------|----------|----------|
| `/register` | f-string 拼接 INSERT 语句 | 改为 `?` 占位符传参 |
| `/search` | f-string 拼接 LIKE 语句 | 改为 `?` 占位符传参 |
